# Data Analytics Internship - Task 1

## Objective
The objective of this task is to perform data cleaning and data wrangling on a raw dataset to make it analysis-ready.

## Tasks Performed
- Understood the dataset structure
- Created a Data Dictionary
- Identified missing values
- Removed duplicate records
- Standardized data formatting
- Created a new calculated column
- Prepared a cleaned dataset for analysis

## Files Included
- original.csv → Raw dataset
- cleaned_dataset.csv → Cleaned dataset
- Task1_Workbook.xlsx → Workbook with Data Dictionary and cleaning work

## Tools Used
- Microsoft Excel
- GitHub

## Outcome
Successfully transformed raw data into a clean and structured dataset ready for further analysis.
